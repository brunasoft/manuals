Manual: Incluir Mais de um Empréstimo Consignado

Guia completo para adicionar múltiplos empréstimos no sistema eConsignado

Visão Geral

Este manual explica o processo para incluir mais de um empréstimo consignado para um funcionário no sistema eConsignado, seguindo as diretrizes do eSocial.

Evento Padrão: O eConsignado utiliza o Evento 66 – "Desconto de Empréstimo Consignado" como padrão.

Como Funciona o Desconto de Múltiplos Empréstimos

Observação Importante: O desconto sempre respeitará o limite máximo de 35% da margem consignável.

O sistema realiza automaticamente o cálculo das bases e aplica os descontos na seguinte ordem:

1. Primeiro Empréstimo

Primeiro, será efetuado o desconto referente ao primeiro evento criado.

2. Segundo Empréstimo

Caso ainda exista saldo disponível, o sistema descontará o valor do segundo evento.

3. Processo Contínuo

Esse processo continuará sucessivamente até atingir o limite da margem.

Atenção: Se o saldo não for suficiente, o sistema efetuará um desconto parcial do valor do empréstimo. Em situações em que não houver saldo disponível, nenhum desconto será aplicado.

Passo a Passo para Gerar Mais de um Consignado

1

Acessar o Módulo de Empréstimo Consignado

No sistema, navegue até: Cadastros > Funcionário > Empréstimo Consignado

Figura 1: Caminho para acessar o módulo de Empréstimo Consignado

2

Selecionar o Funcionário

Selecione o funcionário desejado na lista e clique em "Novo" para criar um novo empréstimo.

Figura 2: Seleção do funcionário e botão "Novo"

3

Preencher Informações do Empréstimo

Insira as informações do empréstimo consignado e, no campo "Código do Evento", informe o código do evento que você já criou e configurou previamente.

Figura 3: Formulário de empréstimo com campo "Código do Evento"

4

Gravar os Dados

Após preencher todas as informações, grave os dados. O sistema criará automaticamente os eventos fixos correspondentes aos devidos recibos, permitindo que você comece a utilizar o módulo imediatamente.

5

Repetir o Processo

Repita o mesmo processo para cada empréstimo que o funcionário possuir.

Informações Importantes

Atenção: O eSocial não permite o uso do mesmo evento dentro de um mesmo recibo. Portanto, se o funcionário possuir, por exemplo, três empréstimos, e o sistema já possui um evento padrão, será necessário criar mais dois eventos.

Reutilização de Eventos: Os eventos criados podem ser reutilizados para outros funcionários. Basta criar apenas os que ainda não existirem, lembrando sempre de configurá-los corretamente e enviá-los ao eSocial.

Para incluir mais de um empréstimo, o usuário deverá:

  • Criar um novo evento idêntico ao Evento 66
  • Configurá-lo adequadamente
  • Enviá-lo ao eSocial

Junto a este documento será encaminhado o passo a passo completo sobre como criar e enviar um evento ao eSocial.
